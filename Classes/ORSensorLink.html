ORSensorLink.m

Overview

Represents a link between a Sensor and a property it updates. This includes the widget and the name of the property.

It includes an optional sensor states mapping to should be used to translate the sensor value before assignment to the Component property.

– initWithComponent:propertyName:sensorStatesMapping:

Initializes the SensorLink with the provided information. No check is done on that information (e.g. that the property exists on the component).
